Start을 누르면 다음 문단을 실행한다.
{
        시계의 값은 "000200"을 참조한다.
        자극그림의 화면감춤을 거짓으로 설정한다.
        최종결과의 화면감춤을 참으로 설정한다.

        점수는 0을 참조한다.
        총자극은 0을 참조한다.
        총정답은 0을 참조한다.
        정답버튼은 ""을 참조한다. 
        입력버튼은 ""을 참조한다.
        조건난수값은 0을 참조한다.
        자극난수값은 0을 참조한다.
      
        시계의 시작을 실행한다.
        총시간은 시계의 값('초')을 참조한다.
        폼의 시간이벤트(1, 1000)을 설정한다.

        시작시간기록을 실행한다.
        자극제시를 실행한다.
}


폼의 시간이벤트를 실행하면 다음 문단을 실행한다(이벤트번호).
{
        이벤트번호가 1이면 다음 문단을 실행한다.
        {
                임시는 시계의 값('초')을 참조한다.                
                상태의 값은 식((임시/총시간) * 100)을 참조한다.
        }

        이벤트번호가 3이면 다음 문단을 실행한다.
        {
                폼의 시간이벤트(3)를 해제한다.
                자극제시를 실행한다.
        }

}

시계의 타이머를 종료하면 다음 문단을 실행한다.
{
        폼의 시간이벤트(1)을 해제한다.
        폼의 시간이벤트(2)을 해제한다.
        폼의 시간이벤트(3)을 해제한다.
        게임오버를 실행한다.
}

Reset을 누르면 다음 문단을 실행한다.
{
        시계의 중지를 실행한다.
        폼의 시간이벤트(1)을 해제한다.
        시계의 값은 "000200"을 참조한다.
        상태의 값은 100을 참조한다.
}

문단 게임오버
{
        자극그림의 화면감춤을 참으로 설정한다.
        반응의 화면감춤을 참으로 설정한다.
        그림1의 화면감춤을 참으로 설정한다.
        그림2의 화면감춤을 참으로 설정한다.
        최종결과값은  식((총정답/총자극)*100)를 참조한다.       
        최종결과값은  최종결과값을 반올림(3)하여 참조한다.
        최종결과는 식("최종정답률  " + 최종결과값 + " %")를 참조한다.
        최종결과의 화면감춤을 거짓으로 설정한다.
}

문단 시작시간기록
{
        시작시는 현재시간('시')을 참조한다.
        시작분은 현재시간('분')을 참조한다.
        시작초는 현재시간('초')을 참조한다.
        시작밀리초는 현재시간('밀리초')을 참조한다.

        시작밀리초의 자릿수가 2 이하이면 다음 문단을 실행한다.
        {
                다음 문단을 반복 실행한다.
                {
                        시작밀리초는 식("0" + 시작밀리초)를 참조한다.
                        시작밀리초의 자릿수가 3이면 다음 문단을 실행한다.
                        {
                                시작밀리초는 시작밀리초를 참조한다.
                                시작시간은 식((시작시 * 3600000) + (시작분 * 60000) + (시작초 * 1000) + 시작밀리초)을 참조한다.
                                문단의 실행을 종료한다.
                        }
                }
        }

        시작시간은 식((시작시 * 3600000) + (시작분 * 60000) + (시작초 * 1000) + 시작밀리초)을 참조한다.
}

{

        자극그림경로는 식("@Path:\\Fix.png")을 참조한다.

문단 자극제시
{        
        시작시간기록을 실행한다. 
        
        반응의 화면감춤을 참으로 설정한다.
        규칙그림의 화면감춤을 거짓으로 설정한다.     
        
        조건난수값은 난수(1, 4)를 참조한다.
        조건그림경로는 식("@Path:\\R_"+조건난수값+".png")을 참조한다.
        규칙그림의 배경그림경로는 조건그림경로를 참조한다.

        자극난수값은 난수(1, 64)를 참조한다.
        자극그림경로는 식("@Path:\\"+자극난수값+".png")을 참조한다.
        자극그림의 배경그림경로는 자극그림경로를 참조한다.

        조건(조건난수값=1 또는 조건난수값=2)을 만족하면 그림버튼보이기를 실행하고
        그 외에는 색상버튼보이기를 실행한다.

        총자극은 이 식(총자극+1)을 참조한다.
}

문단 정답비교
{
        조건(정답버튼=입력버튼)을 만족하면 다음 문단을 실행한다.
        {
                총정답은 식(총정답+1)을 참조한다.
                점수는 식(총정답*5)를 참조한다.
                반응의 화면감춤을 거짓으로 설정한다.

                반응의 배경그림경로는 반응그림경로를 참조한다.  

                빈화면을 실행한다.               
        }
        조건(정답버튼<>입력버튼)을 만족하면 다음 문단을 실행한다.
        {
                반응의 화면감춤을 거짓으로 설정한다.
                반응그림경로는 식("@Path:\\Wrong.png")을 참조한다.
                반응의 배경그림경로는 반응그림경로를 참조한다.                 
        }
}
